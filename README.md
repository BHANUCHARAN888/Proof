# 🎓 Student Management System (Python + MySQL)

## 📌 Overview

The Student Management System is a simple backend-based CRUD application built using Python and MySQL.  
It is designed to manage student records efficiently by allowing users to add, view, update, and delete student information from a database.

This project was built as part of my learning journey in Python backend development and database integration.

---

## 🎯 Purpose of the Project

While learning Python and databases, I wanted to move beyond theory and build something practical.

This project helped me understand:
- How real-world applications interact with databases
- How CRUD operations work behind the scenes
- How to structure Python code for modular development

It is a stepping stone toward my goal of becoming an AI/Data Engineer.

---

## ⚙️ Features

- ➕ Add new student records  
- 📄 View all student details  
- 🔍 Search student by ID  
- ✏️ Update student information  
- ❌ Delete student records  
- 🛡️ Basic input validation and error handling  

---

## 🧰 Tech Stack

- Python 🐍
- MySQL 🗄️
- MySQL Connector for Python

## 🚀 How to Run the Project

1️⃣ Clone the repository
git clone https://github.com/BHANUCHARAN888/Proof.git

2️⃣ Navigate to project folder
cd student-management-system

3️⃣ Install dependencies
pip install mysql-connector-python

4️⃣ Setup MySQL database
Create a database named student_db
Create a table students with required fields

5️⃣ Run the application
python main.py
